# Dementia Progression Survival Analysis
### Overview
This project analyzes factors associated with progression from mild cognitive impairment (MCI) to dementia using survival analysis techniques.
### Methods
Cox proportional hazards model
Kaplan-Meier curves
Model diagnostics and assumption testing
### Tools
R
survival package
tidyverse
### Key Findings
Identified significant predictors of progression risk
Demonstrated application of time-to-event modeling in clinical data
### Files
code.Rmd: sample analysis code for a quick overview of the full analysis. Includes data cleaning, major parts of the model building process, and shortened results and discussion. If curious about the full process and technical details, see Full_report
Summary_Report.pdf : Markdown output of the code.Rmd
outputs/: visualizations and results
Full_Report.pdf : The complete submission for this project. Includes a more detailed description of the dataset, methods, and results using scietific paper formating
