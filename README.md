# Dementia Progression Survival Analysis
### Overview
This project analyzes factors associated with progression from mild cognitive impairment (MCI) to dementia using survival analysis techniques.
### Methods
Cox proportional hazards model
Kaplan-Meier curves
Model diagnostics and assumption testing
### Tools
R
survival package
tidyverse
### Key Findings
Identified significant predictors of progression risk
Demonstrated application of time-to-event modeling in clinical data
### Files
analysis.R: full analysis code
outputs/: visualizations and results
### Notes
This project was completed as part of a graduate biostatistics program.
